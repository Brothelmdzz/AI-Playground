# AI 玩具创意库

> 灵感来源: [Silicon Rider Bench](https://github.com/KCORES/silicon-rider-bench) - 外卖骑手 AI Agent 基准测试
> 核心思路: 真实场景模拟 + 工具调用 + 资源约束 = AI 决策能力测试

---

## ⭐ Top 3 优先项目

### 1. AI 侦探 (Detective Agent)
- **场景**: 案件侦破
- **核心挑战**: 证据收集、嫌疑人审讯、逻辑推理
- **亮点**: 推理链可视化，天然展示 LLM 推理能力
- **状态**: 待规划

### 2. AI 狼人杀 (Werewolf Agent)
- **场景**: 多 Agent 博弈
- **核心挑战**: 谎言检测、身份推理、说服他人
- **亮点**: 多 Agent 交互，社交推理，容易出「名场面」
- **状态**: 🎯 下一个深入研究

### 3. AI 急诊室 (ER Triage Agent)
- **场景**: 医院急诊 triage
- **核心挑战**: 有限床位/医生，患者严重程度不同，动态涌入
- **评估维度**: 存活率、等待时间、资源利用率
- **亮点**: 道德困境 + 资源调度，测试 AI 价值观
- **状态**: 待规划

## 🔧 通用设计模式 (从 Silicon Rider Bench 提炼)

1. **确定性环境**: 使用 seed 保证可复现性
2. **工具调用接口**: 标准化 API，兼容 OpenAI SDK
3. **多维评估指标**: 不只看结果，还看过程（API 调用规范性等）
4. **可视化界面**: 终端 + Web 双模式
5. **难度分级**: 教程关卡 → 完整基准测试
